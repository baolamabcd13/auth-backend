# Express TypeScript Auth Boilerplate

Boilerplate code cho Express.js + TypeScript với authentication system.

## Tính năng

- 🔐 Authentication với JWT
- 👥 User roles (Admin/Editor)
- 📨 Invite code system
- 🔑 Password reset
- 🛡️ Security middlewares
- 📝 Winston logging
- ✅ Input validation
- 🚦 Rate limiting
- 💾 MongoDB + Mongoose
